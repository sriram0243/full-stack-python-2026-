# 💻 Mini Web Projects Collection

This repository contains a collection of beginner-friendly web projects built using HTML, CSS, JavaScript, and Python (Flask). These projects are simple, easy to understand, and perfect for learning basic development concepts.

---

## 📌 Projects Included

### 1️⃣ Student Name Entry System

* Add student names through a web form
* Store data in a database (SQLite)
* Display all student records
* Built using **Flask + HTML**

---

### 2️⃣ Simple Calculator

* Perform basic arithmetic operations (+, −, ×, ÷)
* User-friendly interface
* Built using **HTML, CSS, JavaScript**

---

### 3️⃣ Dancing Robot 🤖

* Fun animated robot
* Uses CSS animations or JavaScript
* Demonstrates basic animation concepts

---

### 4️⃣ Simple Form Fill Website

* User input form (name, email, etc.)
* Form validation
* Clean and simple UI
* Built using **HTML & CSS**

## ▶️ How to Run

1. Install Flask:

```
pip install flask
```

2. Run the application:

```
python app.py
```

3. Open browser:

```
http://127.0.0.1:5000
```
